A standard Micro Servo Motor

## Pin names

| Name | Description           |
| ---- | --------------------- |
| PWM  | Servo control signal  |
| V+   | Positive voltage (5V) |
| GND  | Ground                |

## Attributes

| Name      | Description                              | Default value |
| --------- | ---------------------------------------- | ------------- |
| horn      | Horn type: "single", "double" or "cross" | "single"      |
| hornColor | The color of the servo's horn            | "#ccc"        |
